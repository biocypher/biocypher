## Purpose

The purpose of this documentation is to provide a comprehensive guide to understanding and working with knowledge graphs within the context of BioCypher. It aims to explain the fundamental concepts of knowledge graphs and how they are represented in BioCypher, without prescribing specific step-by-step actions (those live in the Tutorials and How-to guides).

If you are following a tutorial and want to dig deeper into the “why” behind a concept, this is the place to look.
